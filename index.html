Chartr

Chartr is a charting library built using the MooTools javascript library.

Examples

There are currently 3 types of charts available:

  • Line chart
  • Bar chart
  • Pie chart

Download

You can download this project from GitHub in either Zip or Tarball format

Contribute

Get involved in the project to make it better. You can clone the project on GitHub:

$ git clone git://github.com/ryanmitchell/chartr.git
